A **cloud development environment**. One `app` container builds from git and
idles in dev mode so you can attach a remote IDE (VS Code Remote / JetBrains
Gateway), run the dev server inside it, and reach PostgreSQL, object storage,
Meilisearch and Mailpit by hostname — no local toolchain needed. Lightweight
plan. Pick this to develop ZRNO entirely in the cloud.
